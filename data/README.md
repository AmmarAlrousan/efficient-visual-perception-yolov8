## Dataset

This project uses a dense retail shelf dataset for object detection experiments.

Due to licensing restrictions, the dataset is not included in this repository.

### Dataset Used
- **SKU110K**: A large-scale dataset for dense object detection in retail environments.

### Access
The dataset can be obtained from the official source:
See `dataset_link.txt` for details.

### Notes
All experiments were conducted on the original dataset without data augmentation beyond standard YOLO preprocessing.
